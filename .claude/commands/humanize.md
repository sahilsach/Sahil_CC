# Humanize

Rewrite a document to sound more human and less like AI wrote it.

## Instructions

1. Ask the user which file they want to humanize (or use the file from recent conversation context)
2. Read the file
3. Read the writing style guide at `templates/writing-style.md`
4. Rewrite the content following all the rules in the style guide
5. Save the rewritten version back to the same file

## Key Transformations

Apply these changes:
- Replace jargon with plain language
- Convert passive voice to active voice
- Shorten sentences that are too long
- Cut filler words and phrases
- Add specific examples where things are vague
- Make it sound like something a human would say out loud

## Important

- Keep the same structure and sections
- Don't change the meaning, just the tone
- Preserve any data, numbers, or specific facts
- If something is unclear in the original, flag it rather than guessing
